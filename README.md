# DotSize

照片尺寸工具 - 纯 Web 端照片处理工具

## 功能特性

- 📷 相机拍照
- 📁 相册导入
- ✂️ 智能裁剪
- 📏 多种尺寸模板
- 💾 高质量导出
- 📤 分享功能
- 📋 历史记录

## 技术栈

- Vue 3 + TypeScript
- Vite
- Pinia
- Tailwind CSS

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 测试
npm test
```
